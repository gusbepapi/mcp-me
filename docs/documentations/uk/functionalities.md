# Функціонал

## v1.0.0

- Файл `README.md` було перекладено кількома мовами;
- Файли в папці `docs` було перекладено кількома мовами;
- Сторінку аналізу читабельності для ATS було перекладено кількома мовами;
- Створено лексикон термінів (тематичний словник) для резюме у форматі CSV для генерації резюме однією або кількома різними мовами;
- Інструменти (`tools`) та *промпти* (команди) було додано та перекладено кількома мовами;
- Додано багатомовну модель для інтерфейсу MCP Inspector;
- Додано та згенеровано інструкції для користувачів-початківців MCP `mcp-me`:
  - через Pandoc:
    - для офлайн-HTML, оптимізованого та ідеального для будь-якого пристрою;
  - через Typst:
    - для PDF у форматі A6, що ідеально підходить для друку;
    - для PDF з нескінченною висотою та шириною формату A4, що ідеально підходить для комп’ютерів та планшетів;
    - для PDF з нескінченною висотою та шириною 480 пікселів, що ідеально підходить для будь-якого мобільного телефону;
- Інструкції було перекладено кількома мовами;
- Проєкт MCP `mcp-me` було реструктуризовано та реорганізовано, щоб користувачам було простіше створювати власні теми, дотримуючись системи дизайну Pandoc Markdown, $\LaTeX$ та Typst із конвенціями та філософіями BEM, IBM Carbon та W3C DTCG.

* **Генератори PDF на різних движках рендерингу:** Ви можете генерувати свої документи за допомогою LaTeX, Pandoc із Puppeteer, ReportLab PDF Toolkit, Typst та/або WeasyPrint. Система підтримує створення:
  - **Функціональних резюме:** адаптованих під різні мови та візуальні теми;
  - **Модульних резюме:** структурованих під різні кар’єрні напрямки, мови та теми;
  - **Гібридних резюме:** що поєднують понад два кар’єрні напрямки в одному документі з можливістю налаштування під різні мови та теми.
* **Мультитематична архітектура:** Це дозволяє вам вибирати одну або кілька тем для ваших функціональних, модульних та/або гібридних резюме, або навіть створювати власні теми для своїх резюме.
* **Модульна архітектура:** Ви зможете вибрати лише один кар’єрний напрямок для резюме — витягуючи лише релевантну інформацію про професійний досвід, академічну освіту, навички, курси та сертифікати — або два чи більше кар’єрних напрямків для різних резюме чи для одного й того самого резюме, базуючи релевантність на цих конкретних напрямках.
* **Багатомовність:** Ви можете створити папку `i18n` для побудови ключів та їх перекладу в YAML, щоб генерувати резюме однією, двома або більше іноземними мовами.
* **Підтримка MCP для Docker-образів та `distrobox`:** Ви зможете використовувати Docker-образи Густаво Папі для цього MCP без необхідності локально встановлювати безліч зайвих пакетів, що запобігає засміченню вашої системи або зловживанню місцем на диску (жорсткому диску/SSD).
* **Аналіз читабельності для ATS:** Ви можете згенерувати сторінку дашборду ATS для перевірки оцінки та точності рядків (*strings*) кожного PDF-файлу, створеного за допомогою іншого движка рендерингу. Це дозволяє виявити помилки, які призводять до вашого відхилення штучним інтелектом на платформах з працевлаштування, що використовують системи ATS з алгоритмами для фільтрації кандидатів.
